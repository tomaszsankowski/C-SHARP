# TcpCarTuningClientServer

A console application featuring a TCP Client and Server that communicate by sending a `Car` object serialized as JSON.

## Features
- TCP Client and Server communication.
- Serializes and deserializes a `Car` object using JSON.
- Simulates a car tuning service.

## How to Run
1. Open the `TcpCarTuningClientServer.sln` in Visual Studio.
2. Build and run the project.
