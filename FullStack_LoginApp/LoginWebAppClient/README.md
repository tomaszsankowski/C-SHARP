# LoginWebAppClient

An Angular client application that interacts with the `LoginWebApp` API.

## Features
- Interacts with the `LoginWebApp` API.
- Provides a user-friendly Angular interface.
- Uses Angular for the client-side application.

## How to Run
1. Open the `LoginWebAppClient` directory in a terminal.
2. Run `npm install` to install dependencies.
3. Run `ng serve` to start the application.
