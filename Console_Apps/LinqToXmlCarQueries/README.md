# LinqToXmlCarQueries

A console application that uses LINQ to XML and XPath to query and serialize a collection of Cars and Engines.

## Features
- Uses LINQ to XML to query a collection of Cars and Engines.
- Uses XPath to query XML data.
- Serializes and deserializes XML data.

## How to Run
1. Open the `LinqToXmlCarQueries.sln` in Visual Studio.
2. Build and run the project.
