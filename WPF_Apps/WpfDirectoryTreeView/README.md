# WpfDirectoryTreeView

A WPF application that opens a directory, displays its structure in a TreeView, and allows creating or deleting files and folders.

## Features
- Opens a directory and displays its structure in a TreeView.
- Allows creating and deleting files and folders.
- User-friendly WPF interface.

## How to Run
1. Open the `WpfDirectoryTreeView.sln` in Visual Studio.
2. Build and run the project.
