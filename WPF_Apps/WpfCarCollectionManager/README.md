# WpfCarCollectionManager

A WPF application for managing a collection of Cars and Engines, featuring XML serialization, sorting, and searching functionalities.

## Features
- Manages a collection of Cars and Engines.
- Serializes and deserializes the collection using XML.
- Provides sorting and searching functionalities.
- User-friendly WPF interface.

## How to Run
1. Open the `WpfCarCollectionManager.sln` in Visual Studio.
2. Build and run the project.
