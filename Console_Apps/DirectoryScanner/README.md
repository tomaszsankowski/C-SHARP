# DirectoryScanner

A console application that scans a directory, displays its contents, finds the oldest file, and serializes/deserializes a collection using `BinaryFormatter`.

## Features
- Scans a specified directory and displays its contents.
- Finds and displays the oldest file in the directory.
- Serializes and deserializes a collection using `BinaryFormatter`.

## How to Run
1. Open the `DirectoryScanner.sln` in Visual Studio.
2. Build and run the project.
3. Provide a directory path as a command-line argument.
